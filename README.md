# Lakshmi Narasimha Portfolio

A high-end "Dark Tech Editorial" portfolio built with React, Tailwind CSS, and Framer Motion.

## Features
- ⚡ **Vite-powered** React application
- 🎨 **Custom Design System**: Dark background, electric cyan and amber accents
- ✨ **Animations**: Framer Motion staggered reveals, typing effects, and scroll-linked progress
- 📱 **Responsive**: Mobile-first grid layouts
- 📄 **Resume Integration**: Direct link to your resume

## Getting Started

1. **Prerequisites**: Ensure you have [Node.js](https://nodejs.org/) installed.
2. **Setup**:
   - Rename the `Public` folder to `public` (lowercase) so Vite can serve your images.
   - Run the following commands in your terminal:
   ```bash
   npm install
   npm run dev
   ```
3. **Build for Production**:
   ```bash
   npm run build
   ```
